Journey

Dalton Turner

Overview

Journey is a 2D platformer about a professor that explores caves and defeats enemies within them. The original idea came from the book: "Journey to the Center of the Earth" by Jules Verne.

Development

Coding was done in Java 1.8 with no external libraries. Github and SourceTree were used as the version control system and repository. Eclipse Luna was used as the Java development environment.


Game Mechanics

Combat

Defeating enemies involves jumping on top on them while not getting damaged by them


Climbing

Rope is used to climb up walls too high to jump


Health

Enemies have a chance to drop hearts, which can heal the player
