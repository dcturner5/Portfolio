Project Red

Dalton Turner

Overview

Project Red is a multiplayer Pokémon clone designed to be played in a web browser. The client-side game logic and engine were built with JavaScript, while the server-side was built with NodeJS and websockets. The demo on this page is the single player version since the server is no longer hosted online.

Controls

W: Up, S: Down, A: Left, E: Right, Space: Shoot
